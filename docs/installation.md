# Installation

## Requirements

- Python 3.8 or higher
- Qiskit 0.45.0 or higher

## Install from PyPI

```bash
pip install delegated-qc
```

## Install from source

```bash
git clone https://github.com/mtp354/delegated_qc.git
cd delegated_qc
pip install -e .
```

## Development installation

```bash
git clone https://github.com/mtp354/delegated_qc.git
cd delegated_qc
pip install -e .[dev]
```
